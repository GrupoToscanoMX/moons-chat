# Moons Chat

Chat en tiempo real utilizando las tecnologias:

- NodeJS
- React
- Socket.io

La aplicacion funciona localmente en los puertos 5000 (backend) y 3000 (cliente)

## Instalacion

Instalar las dependencias e iniciar el servidor backend

```sh
cd backend
npm install
npm start
```

Instalar las dependencias del cliente e iniciar la aplicacion

```sh
cd frontend
npm install
npm start
```
